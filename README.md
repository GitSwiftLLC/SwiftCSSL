# SwiftCSSL
Swift package for OpenSSL with prebuilt XCFramework

## Installation

Use the Swift Package Manager to install SwiftCSSH with OpenSSL.

```swift
dependencies: [
    .package(url: "https://github.com/GitSwiftLLC/SwiftCSSH.git", from: "1.11.1")
]
```
